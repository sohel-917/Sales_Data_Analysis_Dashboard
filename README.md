# Sales Data Analysis Dashboard

## Project Overview

This project analyzes store sales data to discover business insights using Python.

## Objectives

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights
- Dashboard Creation

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- VS Code

## Dataset

Store Sales Forecasting Dataset from Kaggle.
