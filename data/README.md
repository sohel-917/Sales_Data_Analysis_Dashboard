# Dataset

The dataset is not included in this repository because GitHub has a file size limit of 100 MB.

You can download the dataset from Kaggle:

https://www.kaggle.com/competitions/store-sales-time-series-forecasting/data

After downloading, place these files inside the `data/` folder:

- train.csv
- stores.csv
- oil.csv
- transactions.csv
- holidays_events.csv
